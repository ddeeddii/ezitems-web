# ezitems-web
Web app to create simple item rename/resprite mods for Repentance.

Upgrade/Alternative to `isaac-ez-name-changer`.

## Usage
### Option 1
Run the GitHub pages in your preferred browser

### Option 2
Clone the repo and start a local server
